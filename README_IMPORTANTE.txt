IMPORTANTE:

1. ARRASTE A PASTA "PASTA_LIMPA_PRA_UPLOAD" PARA O VERCEL.
   NÃO arraste a pasta antiga.
   NÃO arraste do VS Code.
   Arraste diretamente do Finder / Área de Trabalho.

2. Se o erro 404 persistir:
   - Vá nas configurações do projeto no Vercel (Settings -> Build & Development).
   - Verifique se "Output Directory" está configurado como 'dist'.
   - Se "Framework Preset" estiver como "Other", mude para "Vite".

Esta pasta contém apenas os arquivos essenciais. Se der erro de "muitos arquivos", é porque você está arrastando a pasta errada.
